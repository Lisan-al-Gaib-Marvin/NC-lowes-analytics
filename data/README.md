# Data Directory

The CSV files used in this project are not tracked in Git (they contain
data collected from public sources that should be regenerated independently).

## How to Recreate the Data

### data/raw/

| File | Source | How to Get It |
|------|--------|---------------|
| `lowes_financials_complete.csv` | SEC EDGAR 8-K/10-K filings | Extract from Lowe's quarterly earnings press releases (CIK: 0000060667) |
| `lowes_nc_reviews.csv` | Google Maps | Search each NC city from lowes.com/Lowes-Stores/North-Carolina/NC |
| `lowes_sustainability.csv` | Lowe's CSR Reports | Extract from annual Corporate Responsibility Report PDFs at corporate.lowes.com |

### data/clean/

| File | Source |
|------|--------|
| `clean_financials.csv` | Generated by `sql/cleaning_tables.sql` from the financials staging table |
| `clean_reviews.csv` | Generated by `sql/cleaning_tables.sql` from the reviews staging table |

These clean CSVs are what Tableau and Power BI connect to.
